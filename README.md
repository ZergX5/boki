# boki

<p>An experimental drafting tool for League of Legends.</p>

<h2>🚀 Features</h2>
<ul>
  <li>⚡ Fast calculations based on drafts</li>
  <li>🔌 Updated every patch</li>

  <li>🌍 Not dependant on server! </li>
</ul>
